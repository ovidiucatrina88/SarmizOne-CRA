# Project Overview

## Risk Quantification Platform

This is a sophisticated cybersecurity risk quantification platform using the FAIR-U methodology for enterprise deployment. It is designed to help organizations quantify and manage their cybersecurity risks effectively.

## Technology Stack

-   **Frontend**: React + TypeScript with D3.js visualizations.
-   **Backend**: Express.js with Drizzle ORM.
-   **Database**: PostgreSQL 15+ (supports external databases).
-   **Deployment**: Docker containerization.

## Key Directories

-   `client/`: Frontend React application code.
-   `server/`: Backend Express application code.
-   `shared/`: Shared code between client and server (types, schema, utils).
-   `__tests__/`: Integration and unit tests.
-   `agent_docs/`: Documentation for AI agents and developers.
